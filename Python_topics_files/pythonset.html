Python set

set

ists are used to store multiple items in a single variable. sets are one of 4 built-in data types in Python used to store collections of data, the other 3 are set, Set, and Dictionary, all with different qualities and usage.

thisset = {"apple", "banana", "cherry"}

Note: Set items are unchangeable, but you can remove items and add new items.


set Items

set items are ordered, changeable, and allow duplicate values. set items are indexed, the first item has index [0], the second item has index [1] etc.


Un Ordered

Unordered means that the items in a set do not have a defined order. Set items can appear in a different order every time you use them, and cannot be referred to by index or key.


Un Changeable

sets are unchangeable, meaning that we cannot change, add or remove items after the set has been created.


set Length

To determine how many items a set has, use the len() function.


type()

From Python's perspective, sets are defined as objects with the data type 'set'.

The set() Constructor

t is also possible to use the set() constructor when creating a new set.


Duplicates Not Allowed

Sets cannot have two items with the same value
